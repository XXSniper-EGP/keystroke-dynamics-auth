# Keystroke Dynamics Authentication

Web app for user authentication using keystroke dynamics (typing rhythm) as an extra security layer on top of a traditional password-based login, built with Flask.

## Features
- User registration and login with traditional password.
- Keystroke data collection and training for a fixed passphrase.
- Verification step that checks typing pattern after successful password login.

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## How to Run
1. Create and activate a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Run `python app.py` and open the shown URL in your browser.
